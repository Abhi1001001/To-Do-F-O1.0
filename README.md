<h1>Frontend --> To-Do Application</h1>
<p>This is the frontend of the To-Do application. It is built using React and styled with Tailwind CSS. The UI is clean, simple, and easy to use. It contains pages for Login, Signup, Forget Password, Reset Password, and the main To-Do dashboard.</p>
<hr>
<h2>Features ---></h2>
<ul>
  <li>Beautiful responsive UI using Tailwind CSS</li>
  <li>Login, Signup, Forget Password, Reset Password pages</li>
  <li>Modern To-Do page with add, edit, delete, and complete actions</li>
  <li>Modal popup for editing tasks</li>
  <li>Separate components for better structure</li>
  <li>Ready to connect with backend API</li>
</ul>
<hr>
<h2>How to Install</h2>
<pre>
cd frontend
npm install
</pre>
<hr>
<h2>How to Run the Project</h2>
<pre>
npm run dev
</pre>
<p>The app will run at:</p>
<p><strong>https://to-do-app-min.netlify.app/</strong></p>
<hr>
<h2>API Configuration</h2>
<p>You can set the backend API URL inside:</p>
<pre>.env</pre>
<p>Example:</p>
<pre>
VITE_API_URL=https://to-do-b-o1-0.onrender.com
</pre>
